# AirportCEOUndergroundStructures
A code based mod for Airport CEO that allows some buildings to be built underground.

**Requires [Airport CEO Mod Loader](https://steamcommunity.com/workshop/filedetails/?id=3109136766) to function**.

Currently allows you to place the following buildings underground

* Airport Police Station
* Emergency Response Station
* Catering Depot
* Waste Depot

Each of these can be individually enabled or disabled to be built underground using the F1 menu.

Code is hosted on [GitHub](https://github.com/baseballlover723/AirportCEOUndergroundStructures) with the MIT license. Feedback and contributions or any other comments are welcome.

Icon is generated by ChatGPT cause I don't really care about the icon. If anyone wants to give me a different one, I'll use that one instead.

## Releasing

Run `dotnet build -c Release`

Then copy over the dll and pdb file to `%AppData%\Apoapsis Studios\Airport CEO\Mods\AirportCEOUndergroundStructures\plugins`.
Then load up Airport CEO and upload the local mod to Steam.
